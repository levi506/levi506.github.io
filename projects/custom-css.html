Replacing Discord Overlay Font with Splatoon Font

To apply this put it in the Custom CSS Section while adding or editing a Discord Overlay Source.

@font-face {
font-family: project-paintball;
src: url(https://levi506.github.io/Paintball_Beta_4a.otf);
}
.voice-container {
font-family: project-paintball;
}
