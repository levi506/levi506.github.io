Open Broadcasting Software CSS

These are just some CSS codes I created to change some web elements in OBS feel free to use them.


Discord Stream Overlay Font

To apply this put it in the Custom CSS Section while adding or editing a Discord Overlay Source.

@font-face {

font-family: Paintball;
src: url(https://levi506.github.io/fonts/Paintball_Beta_4a.otf);
}
.voice-container {
font-family: Paintball;
}

You can download the font for designing any stream assets here.
This font is made possible by theFIZZYnator on the Squidboards.
